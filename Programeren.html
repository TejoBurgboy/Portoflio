Bij het vak Inleiding Programmeren heb ik kennis gemaakt met Javascript. 

Voor de minor "Het internet is stuk maar we gaan het repareren" hebben we veel gewerkt aan de hand van de public stack. We hebben iedere week een een bootcamp gehad om op een bepaalde manier naar de public stack te kijken. Deze bootcamps waren denken over Tech, ICT, juridisch, economie en design.

De Pokémon fighter

Voor de eindopdracht ervan heb ik een Pokemon fighter gemaakt. Ik heb een fighter gemaakt waarin je drie Pokemons kan kiezen en je tegenstander een van de andere Pokemons willekeurig kiest. Ik heb ook gecodeerd dat de tegenstander een willekeurige aanval doet. Als ik extra tijd hiervoor had gehad zou ik een betere healthbar creeëren, meer pokemons toevoegen en meer de pokemon types uitwerken.

Reflectie

Deze opdracht heeft me echt laten inzien dat ik talent heb ik code met hoe makkelijk en gemotiveerd voor deze opdracht was. Ik heb voor deze opdracht zelfs extra code ingevoegd voor het nemen van een snelheid statistiek. Ik ben echt heel trots op het eind product.
