IoT

Voor het vak IoT (Internet of Things) moest ik een IoT product maken van minimaal niveau 2. Het IoT product moest ook een natural, contextual, physical en digital UI bevatten.

Ontwerp

Ik heb gekozen om een slim bed te ontwerpen. Van het slimme bed kan je doormiddel van een physical en digital UI de tempratuur van het bed bepalen, kan je het licht van het bed aan en uit doen door je stem (natural) ook past het licht zich aan het licht buiten aan (contextual) en het bed past zich aan hoe iemand ligt (contextual).

Code

Daarnaast heb ik in Arduino Code geschreven waarmee je de ledstrip van de Arduino kan aanpassen aan hoe laat het is. Hier is de link naar de manual: https://github.com/TejoBurgboy/IOT-Techo/blob/main/Manuel_IOT_Tejo.md.

Reflectie

Bij deze opdracht heb ik leren kijken hoe je op verschillende manieren een geheel kan maken en hoe ik op verschillende manieren iets iets kan veranderen. Ook heb ik gebruik gemaakt van illustrator templates iets wat ik in de toekomst meer wil gebruiken. Gebruikte skills: werken met ledstrip van Arduino en het maken/ontwerpen van een IoT product. Gebruikte tools: Illustrator en Arduino
